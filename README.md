# ERP
Desarrollo ERP Enterprise Resource Planning (Planificación de recursos empresariales de la empresa DECORLATA.

MODULOS A DESARROLLAR 
	• Inventario y Almacén.
	• Ventas y Facturación.
	• Compras.
	• Producción.

INVENTARIO Y ALMACEN
Control de existencias, trazabilidad de productos y gestión de stock.

VENTAS Y FACTURACION
Seguimiento de pedidos, emisión automatica de comprobantes y cotizaciones.

COMPRAS
Gestión de proveedores, ordenes de compra y recepción de mercancias.

PRODUCCION
Planificación y control de la cadena de manufactura.

***********************************************************

LEGUAGES A EMPLEAR

SQL → PostgreSQL
Backend → Python
Frontend:
JavaScript ► Controlar la lógica del navegado.
HTML ► Define la estructura del ERP(los esqueletos de los formularios, las tablas de inventario, listas de ordenes de produccion, etc).
CSS ► Define el diseño visual, colores y acomodo de las pantallas para que los menus sean limpios y faciles de usar.
